# SAMA AI — Workspace

AI-assisted workspace for managing the online presence of **Service and Mercy Alliance (SAMA)**, a Muslim community masjid in Grand Island, NY.

**Website**: https://www.samagi.org
**Instagram**: @sama.grandisland
**Location**: 1822 Huth Rd, Grand Island, NY 14072
